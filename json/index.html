关于 JSON 处理器

格式化 (Prettify): 将混乱的 JSON 字符串按指定的缩进重新排列，极大地提高可读性，方便开发者调试排错。

压缩 (Minify): 移除所有不必要的空白符和换行符，减小体积，适用于网络传输或配置文件部署。

可视化视图: 以树状结构递归展示嵌套的 JSON 对象与数组。您可以一键折叠或展开节点，直观地洞悉数据层级。

转义与去转义: 针对字符串中的特殊字符（如引号、反斜杠）进行处理，方便在代码或配置中嵌套使用 JSON。
